# Avatar Görselleri

Bu klasöre rastgele avatar görsellerini ekleyin.

## Kullanım

- Avatar görselleri rastgele seçilip bildirimlerde gösterilir
- Özel mesajlarda point bildirimlerinde kullanılır
- Seviye atlama bildirimlerinde kullanılır

## Dosya İsimlendirme

- `avatar1.png`, `avatar2.png`, `avatar3.png`, vb.
- `avatar 14.png`, `avatar 15.png` (boşluklu format da desteklenir)
- Herhangi bir isimle ekleyebilirsiniz, bot otomatik olarak bulur

## Desteklenen Formatlar

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
