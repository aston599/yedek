# Üyelik Seviyesi Görselleri

Bu klasöre üyelik seviyesi ikonlarını ekleyin.

## Gerekli Görseller

1. **bronz icon png.png** - Bronz seviye ikonu
2. **gümüş icon png.png** - Gümüş seviye ikonu  
3. **gold icon png.png** - Altın seviye ikonu
4. **plat icon png.png** - Platin seviye ikonu
5. **diamond icon png.png** - Elmas seviye ikonu

## Seviye Eşikleri

- **Bronz:** 0 - 10 KP
- **Gümüş:** 10 - 50 KP
- **Gold:** 50 - 100 KP
- **Plat:** 100 - 500 KP
- **Diamond:** 500+ KP

## Kullanım

Görselleri bu klasöre ekledikten sonra bot otomatik olarak:
- Profilde seviye ikonunu gösterecek
- Point bildirimlerinde seviye ikonunu gösterecek
- Seviye değişikliğinde bildirim gönderecek
