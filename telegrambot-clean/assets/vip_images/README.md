# VIP Görselleri

Bu klasöre VIP seviyesi için özel görselleri ekleyin.

## Kullanım

- VIP seviyesine ulaşan kullanıcılar için özel görseller
- Seviye atlama bildirimlerinde kullanılır
- Profil görsellerinde kullanılabilir

## Dosya İsimlendirme

- İstediğiniz isimle ekleyebilirsiniz
- Bot otomatik olarak rastgele seçer

## Desteklenen Formatlar

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
